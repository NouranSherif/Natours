# Natours // Nature Tours Landing Page 🌲

Natours is a high-end, visually stunning landing page for an imaginary nature tour company. This project was a deep dive into advanced CSS and Sass, focusing on building complex, maintainable, and performant user interfaces without relying on external CSS frameworks.

##  Live Demo
**[View Project Live]** -> (https://natours-omega-tawny.vercel.app/)

---

##  Key Features
* **Custom Grid System:** Built using a specialized float-based grid (later updated to Flexbox/Grid concepts).
* **Advanced Animations:** Includes complex CSS keyframe animations for the header, buttons, and hover states.
* **Pure CSS Components:** Features a "Card Flip" effect, a custom navigation popup, and a video background—all built using **pure CSS (no JavaScript)**.
* **The 7-1 SASS Pattern:** Implemented a professional architecture by splitting styles into 7 different folders (base, components, layout, pages, etc.).
* **Responsive Design:** Utilized mixins for efficient media queries to ensure a flawless experience across mobile, tablet, and desktop.

##  Technical Stack
* **HTML5:** Semantic markup for SEO and accessibility.
* **SASS (SCSS):** Leveraging variables, mixins, nesting, and mathematical functions.
* **NPM Scripts:** Set up a custom build process for compiling, prefixing, and compressing SASS to CSS.
* **BEM Methodology:** Used Block-Element-Modifier to keep the codebase modular and organized.

## Technical Highlights
* **The Checkbox Hack:** Used the checkbox hack to build a fully functional, zero-JS navigation menu.
* **Clip-Path & Shapes:** Implemented advanced `clip-path` properties for the diagonal sections and polygon shapes.
* **Image Floats:** Utilized `shape-outside` for text-wrapping around circular images.

##  Project Context
This project was developed during the **"Advanced CSS and Sass"** course by Jonas Schmedtmann. It serves as proof of my ability to architect large-scale CSS projects and create complex UI interactions using modern styling techniques.
